## Architectural Contracts

1. `app.js` composes modules and owns DOM event wiring; reusable domain logic
   belongs in `src/app-*.js`.
2. Persisted state changes require a migration and a
   `state-migrations-test.mjs` case.
3. New runtime modules are registered once in `module-manifest.js`; boot and
   Service Worker caching consume the same order.
4. Expensive per-game work must be memoized per render and invalidated at the
   top of `render()`.
5. Provider facts remain source-attributed; deterministic local output is the
   fallback when live providers are unavailable.
6. Invisible regression classes become gates in `check.sh` and CI.
