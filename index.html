PSN library import

Paste your npsso cookie to import your PS5/PS4 library automatically.

How to find your NPSSO token
  1. Open my.playstation.com in a browser and sign in.
  2. Press F12 to open DevTools → Application tab.
  3. Under Storage → Cookies, find npsso.
  4. Copy the value (64 characters) and paste below.

The token is only sent to your local companion server (127.0.0.1) — never to any third party.
