# PlaySputnik Product Minimum

## Positioning

PlaySputnik is an AI game companion, not a generic price tracker.

It remembers taste, library, completed games, wishlist, subscriptions, upcoming radar, and buy-later signals so the user can quickly decide what to play, claim, wait for, or buy later.

## Minimum User Value

1. Understand the user's taste through quick choices, ratings, import, or future PSN/library access.
2. Recommend what to play today.
3. Triage new subscription/monthly drops.
4. Maintain wishlist, completed games, library, backlog, and personal ranking.
5. Track future games that match taste.
6. Watch prices only for wishlist and likely-fit games, not the whole store.
7. Keep game cards visual: title, short description, cover/placeholder, atoms, and memory actions.

## First-Run Rule

The user must see value in the first 30 seconds. A perfect profile can come later.

## Data Rule

Taste inference can come from AI and structured atoms. Store, subscription, and price facts need source/freshness metadata.
