# Companion Decision Agenda

## Purpose

The user should not have to inspect every panel. PlaySputnik should turn taste, library, subscription, monthly drops, radar, and price signals into a short agenda.

## Agenda Slots

- Tonight: the strongest play-now recommendation.
- Use access: active, paused, owned, permanently owned, saved, or subscription-accessible game to prioritize before buying more.
- Free trial: a claimable/subscription drop worth testing with a bounded time window.
- Claim only: a free item to add to the library without spending storage or evening time.
- Watch: future/radar item that matches taste.
- Buy later: a price signal, never the primary product promise.

## Product Rule

The agenda is the main companion output. Detailed panels remain available for audit, but the first useful answer should be a compact next-action plan.
